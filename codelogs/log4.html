Code Log 4: CSS Color

In CSS, color is used to control how text and other parts of elements look on the page. Color can make a page easier to read, create contrast, and support a certain vibe or theme. CSS does not only color text—CSS can also color backgrounds and borders. The most common properties you will use are color, background-color, and border-color.

What these CSS properties do

  • color sets the color of the text (the “foreground” text color). If you apply it to a parent element, it usually inherits down to child text unless a child has its own color.
  • background-color sets the color behind the content and padding of an element. This is the “fill” color of the box. Background color does not automatically change text color, so you should make sure your text still has contrast.
  • border-color sets the color of the border. Borders are only visible if you set a border style/width (like border: 3px solid;). You can set one color for all sides or different colors per side.

How CSS accepts color as a value

CSS can accept color values in multiple formats. Some are quick and easy (like named colors), and others are more precise (like hex, RGB, or HSL). Below are examples of different color formats you can use in CSS.

1) Named colors

Named colors are built-in keywords like red, blue, tomato, or rebeccapurple. They are easy to remember, but you’re limited to the names CSS supports.

2) Hexadecimal (hex)

Hex colors start with # and use 6 digits (or sometimes 3). The digits represent red, green, and blue amounts. For example, #ff0000 is pure red, and #000000 is black.

3) RGB

RGB uses rgb(r, g, b) with values from 0 to 255. This is useful when you think in terms of mixing red, green, and blue light. There is also rgba() where the last value controls transparency.

4) Another format: HSL

HSL stands for Hue, Saturation, and Lightness. Many people like HSL because it can be easier to adjust a color by “feel.” You can also use hsla() for transparency.

In real websites, you’ll often combine these formats depending on what you need. If you want exact brand colors, hex is common. If you want to quickly tweak shades or lightness, HSL is super convenient.
